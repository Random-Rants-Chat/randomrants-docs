# `;broadcast` Command

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This command can display a message as the `[Notice]` server message in chat.

# Usage

```messagecommand
;broadcast <Message>
```

# Examples

```messagecommand
;broadcast This is a message from a user
```
