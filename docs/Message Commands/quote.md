# `;quote` Command

:::joke
:::

Makes the `[Notice]` quote the provided message.
If no message was provided, then it would quote the last message the specified username sent.
Quotes like `“the message to quote” -gvbvdxx, 2025`, with the year.

# Usage

```messagecommand
;quote <Username> <Message (Optional)>
```

# Examples

```messagecommand
;quote gvbvdxx Very good thing to quote
```

```messagecommand
;quote gvbvdxx
```

```messagecommand
;quote @self
```
