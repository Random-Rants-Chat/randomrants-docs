# Message Command Docs

_Message Commands_ (or just Commands for short) in **Random Rants +** are commands you can type in your Message box to do various actions.

To see the list of commands, you can type `;help` in your message box.

:::tip
If you set the command permissions to "Everyone", everyone can do commands, but they need Owner Permissions (Called ownership or co-owner), to get all of them.
:::

For getting started, its recommended you view the [`;help`](./help.md) command page.
