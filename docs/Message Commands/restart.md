# `;restart` Command

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This command restarts the room, this causes a reconnection on everyone's screen. This will stop everything in the room.
This also clears all the messages.

# Usage

```messagecommand
;restart
```
