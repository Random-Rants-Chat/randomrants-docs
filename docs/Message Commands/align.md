# `;align` Command

:::joke
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This resets the [`;mildlyInfuriating`](./mildlyInfuriating.md) command for the specified user.

Stops the slow rotation of the screen for the specified user.

# Usage

```messagecommand
;align <Username>
```

# Examples

```messagecommand
;align @all
```

```messagecommand
;align @others
```

```messagecommand
;align gvbvdxx
```
