# `;redirect` Command

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

:::info
This command will or can block user interaction, or make it temporarily tough for the user to use Random Rants +.
:::

This command will switch their current page to the specified URL.

:::warning
As of the time of writing this, we don't have a confirmation message for when this command is ran, so anyone using this could potentially redirect someone to a bad website without them knowing.
:::

# Usage

```messagecommand
;redirect <Username> <URL>
```

# Examples

```messagecommand
;redirect gvbvdxx https://google.com
```
