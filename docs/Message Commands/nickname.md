# `;nickname` Command

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This renames a users display name to something else in that room, until they leave the room.

# Usage

```messagecommand
;nickname <Username> <New Display Name>
```

# Examples

```messagecommand
;nickname gvbvdxx CoolRanter2025
```
