# `;8ball` Command

:::joke
:::

Ask the magic 8-ball a yes-or-no question and it will give you a "wise" answer.

# Usage

```messagecommand
;8ball <question>
```

# Examples

```messagecommand
;8ball will I fail my algebra test?
```

```messagecommand
;8ball is this website cool?
```

```messagecommand
;roll 2d6
```
