# `;me` Command

:::joke
:::

This command broadcasts an "action" message to the room, describing what you are doing.
It's a classic chat feature for role-playing or just being goofy.

# Usage

```messagecommand
;me <message>
```

# Examples

```messagecommand
;me is coding this site
```

```messagecommand
;me is grounded
```
