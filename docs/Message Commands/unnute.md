# `;unmute` Command

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This command unmutes a user who was previously muted with the ;mute command.

# Usage

```messagecommand
;unmute <Username>
```

# Examples

```messagecommand
;unmute Im_CatmanYT
```

```messagecommand
;unmute @all
```
