# `;confetti` Command

:::joke
:::

This shows confetti on the specified user's screen, supposed to be a little reference to that meme that goes _"Yipee!!"_. A non-interaction-blocking joke command.

If no username is provided, this command will do it on everyones screen.

# Usage

```messagecommand
;confetti <Username (Optional)>
```
