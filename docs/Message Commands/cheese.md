# `;cheese` Command

:::joke
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This command starts spawning cheese on the specified user's screen.
When the user clicks the cheese, it says _**+1 CHEESE**_, then disapears. This doesn't mean anything.
The cheese can time out if not clicked in a short period of time.
The cheese will stop spawning after a few seconds.

# Usage

```messagecommand
;cheese <Username>
```

# Examples

```messagecommand
;cheese @all
```

```messagecommand
;cheese @others
```

```messagecommand
;cheese gvbvdxx
```
