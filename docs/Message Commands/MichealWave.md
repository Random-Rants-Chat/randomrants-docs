# `;MichealWave` Command

:::joke
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This shows a Microwave dancing on the specified user's screen. This dancing microwave will teleport to random places very quickly, and the user has to click it or wait 30 seconds after it appears to remove it.

If no user was provided, then it chooses an random user to display it on.

# Usage

```messagecommand
;MichealWave <Username (Optional)>
```

# Examples

```messagecommand
;MichealWave
```

```messagecommand
;MichealWave @all
```

```messagecommand
;MichealWave @others
```

```messagecommand
;MichealWave gvbvdxx
```
