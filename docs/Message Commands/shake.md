# `;shake` Command

:::joke
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

:::info
This command will or can block user interaction, or make it temporarily tough for the user to use Random Rants +.
:::

This makes the sender's screen shake for the specified intensity.

The maximum intensity is 10.

# Usage

```messagecommand
;popcat <Username> <Intensity>
```

# Examples

```messagecommand
;shake gvbvdxx 10
```

```messagecommand
;shake @all 4
```

```messagecommand
;shake @others 5
```
