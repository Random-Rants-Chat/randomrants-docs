# `;funni` Command

:::joke
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

:::info
This command will or can block user interaction, or make it temporarily tough for the user to use Random Rants +.
:::

:::danger
This command can be \_**VERY**\_loud on your speakers and headphones, turn down the volume before someone plays this sound.
:::

Shows a old Gvbvdxx Chat user's OC (called "MiniScratchGamer") laughing on the sender's screen. This also plays a lound Laughing Chihuahua sound.

# Usage

```messagecommand
;funni
```
