# `;importantMessage` Command

:::joke
This is a mix between a joke, and not a joke, usually used for our jokes when we use our site, but it could be used for something serious.
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

Makes the screen of the specified user empty, except the background and a black box.
When they click the black box, it slowly reveals the specified message.
If they don't click it in the period of time, then it just disapears.

# Usage

```messagecommand
;importantMessage <Username> <Message>
```

# Examples

```messagecommand
;importantMessage @all This is a funny joke text
```

```messagecommand
;importantMessage @others This is also a funny joke text
```

```messagecommand
;importantMessage gvbvdxx This is a funny joke text
```
