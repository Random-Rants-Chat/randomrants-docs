# `;flash` Command

:::joke
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

Makes the specified user's background flash white for a breif moment.

# Usage

```messagecommand
;flash <Username>
```

# Examples

```messagecommand
;flash gvbvdxx
```

```messagecommand
;flash @all
```

```messagecommand
;flash @others
```
