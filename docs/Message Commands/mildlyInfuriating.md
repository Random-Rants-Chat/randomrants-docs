# `;mildlyInfuriating` Command

:::joke
:::

:::info
This command is an owner-only command, only people with ownership can use this. (the room owner, and any promoted people)
:::

This starts to tilt the specified user's screen very slowly so that it's not visible, and makes them "infuriated" when they notice something is off.

Can be aligned back using the [`;align`](./align.md) command.

# Usage

```messagecommand
;mildlyInfuriating <Username>
```

# Examples

```messagecommand
;mildlyInfuriating @all
```

```messagecommand
;mildlyInfuriating @others
```

```messagecommand
;mildlyInfuriating gvbvdxx
```
